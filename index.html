Montaje de NAS

He montado un sistema NAS facilmente escalable y sin importar tu hardware, yo use un viejo portatil con una unidad de almacenamiento externa, instale sistema operativo he usado Ubuntu Server. Ademas para garantizar la seguridad se ha configurado el sistema de permisos. Este NAS permite compartir y almacenar archivos sin importar el sistema operativo del cliente.

Objetivo: Crear una solución de almacenamiento compartido segura y accesible desde múltiples dispositivos en una misma red.

Tecnologías Utilizadas

Ubuntu Server Samba Windows

Características Clave

  • Segurización mediante permisos bien estructurados
  • Rendimiento incluso en hardware limitado
  • Facilmente escalable y exportable a hardware mas portente
  • Compatibilidad de cliente con varios sistemas operativos

IA generativa en local

He aprendido a implementar la IA generativa en local, lo que significa que no se necesita acceder a internet, asi mismo poder asegurar que es un sistema mas seguro de cualquier posible ataque o sniffing desde el exterior. Es capaz de aprender de archivos que el usuario suba y generar texto basandose en estos archivos, también opcionalmente generar imagenes, existe la posibilidad de crear usuarios independientes.

Objetivo: Implementar una solución de IA generativa en entorno local, aumenta la productividad mientras que es capaz de operar sin conexión a internet, garantizando la privacidad y seguridad de los datos.

Tecnologías Utilizadas

Linux Ollama Mistral IA generativa Docker IA

Componentes Clave

  • Aumento de productividad
  • Seguridad de los datos y filtraciones
  • Multiusuario simultaneo
  • Sistema sencillo de mantener
  • Generación de contenido texto e imagenes
  • Sistema totalmente en local
  • Capacidad de aprendizaje a partir de archivos subidos por el usuario
